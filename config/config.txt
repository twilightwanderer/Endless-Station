##	(К слову, теоретически можно также уменьшить эффект анти-лаговых эвристик в базовой частоте тиков, установив значение, например, 0.5)

## Раскомментировать, чтобы запретить профилирование пользователям с R_DEBUG, для повышения производительности.
#FORBID_ADMIN_PROFILING

## Раскомментировать, чтобы отключить TTS-сообщения при шепоте, например, радиосообщения или команду whisper. Полезно для снижения нагрузки на TTS.
#TTS_NO_WHISPER

## Ссылка на HTTP сервер, настроенный на сервере. Файл docker-compose находится в tools/tts.
#TTS_HTTP_URL http://localhost:5002

## Токен для предотвращения злоупотребления вашим TTS сервером.
#TTS_HTTP_TOKEN coolio

## Максимальное количество одновременных HTTP-запросов к TTS серверу.
#TTS_MAX_CONCURRENT_REQUESTS 4

## Время ожидания (в секундах) перед таймаутом HTTP-запроса к TTS (0 — бесконечное, не рекомендуется).
#TTS_HTTP_TIMEOUT_SECONDS 30

## Добавить голоса в черный список TTS.
#TTS_VOICE_BLACKLIST Sans Undertale
#TTS_VOICE_BLACKLIST Papyrus Undertale

## Закомментировать, чтобы отключить отправку уведомления toast на хост-сервере при завершении инициализации.
## Даже если включено, уведомление отправляется только если нет подключенных клиентов.
TOAST_NOTIFICATION_ON_INIT

## Выводить ошибки конфигурации в виде runtime ошибок.
CONFIG_ERRORS_RUNTIME

## Ограничение на загрузку файлов клиентом на сервер, в байтах, по умолчанию 0.5 МБ.
UPLOAD_LIMIT 524288

## Ограничение на загрузку файлов админом на сервер, в байтах, по умолчанию 5 МБ.
UPLOAD_LIMIT_ADMIN 5242880

## Раскомментировать, чтобы разрешить админам с +DEBUG запускать профайлер byond-tracy во время раунда.
#ALLOW_TRACY_START

## Раскомментировать, чтобы разрешить админам с +DEBUG ставить в очередь следующий раунд с профайлером byond-tracy.
#ALLOW_TRACY_QUEUE

## TGUI payloads, превышающие лимит 2 КБ для BYOND topic запросов, разбиваются на части примерно по 1 КБ и отправляются последовательно.
## Эта настройка ограничивает максимальное количество частей, которые сервер примет, по умолчанию 32.
TGUI_MAX_CHUNK_COUNT 32

## Раскомментировать и заменить на URL, где пользователи могут подтвердить свои аккаунты BYOND.
#FORUM_LINK_URI

## Пока раскомментировано, инициализация сервера включает генерацию ассетов, полезную для разработки.
## Значительно увеличивает время инициализации, поэтому в продакшене лучше отключить.
GENERATE_ASSETS_IN_INIT
